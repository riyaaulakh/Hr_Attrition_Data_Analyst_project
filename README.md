--HR Attrition Analysis
A small project where I analyzed employee attrition data using Python, PostgreSQL, and Power BI.
-- What this project is about?
I used the IBM HR Employee Attrition dataset to find out why employees leave a company. I cleaned the data in Python, ran some SQL queries in PostgreSQL to answer business questions, and then built a dashboard in Power BI to show the results visually.
--Tools I used
1. Python (pandas)
2. PostgreSQL
3. Power BI
--Key findings

Overall attrition rate is 16.12%
Employees who work overtime leave at a much higher rate (~30%) than those who don't
Sales Representatives and Lab Technicians have the highest attrition among job roles
Most attrition happens in the 26–35 age group
